Registro de Níveis

Flexbox Zombies

Nível 1: Aprendi a usar justify-content: center para alinhar os zumbis no meio do container.

Nível 2: Neste nível, foi necessário usar justify-content: space-between para espaçar os zumbis igualmente entre si. Desafiador por exigir mais observação.

Nível 3: Usar justify-content: space-around foi interessante, pois o espaçamento também é aplicado nas laterais externas, o que muda visualmente o layout.

Flexbox Froggy

Nível 1: Primeiro desafio simples, bastava colocar o sapo na extremidade direita com justify-content: flex-end.

Nível 2: Nesse, alinhar dois sapos exigiu atenção à ordem e posição. Usei justify-content: center.

Nível 6: Foi desafiador por precisar usar justify-content junto com align-items para centralizar o sapo tanto horizontal quanto verticalmente.

Reflexão e Aplicação Prática

  • display: flex — define que o container será flexível
  • justify-content — alinha os itens no eixo horizontal (principal)
  • align-items — alinha os itens no eixo vertical (secundário)
  • flex-start — alinha ao início do container
  • flex-end — alinha ao final do container
  • center — centraliza os itens
  • space-between — espaço igual entre os itens
  • space-around — espaço igual em volta dos itens

Aplicação prática

Se eu fosse criar um menu de navegação responsivo para um site, usaria o Flexbox para alinhar os itens do menu horizontalmente no topo da página. Com justify-content: space-between, poderia distribuir os links igualmente entre a esquerda e a direita. Se quisesse centralizar os itens em um card, usaria justify-content: center e align-items: center para deixá-los perfeitamente centralizados, ideal para layouts de produtos ou posts.
